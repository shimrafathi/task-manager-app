# 📋 Task Manager App

A simple and user-friendly Android Task Manager application that helps users manage daily tasks efficiently. This app allows users to add, update, delete, and track their tasks with a clean and minimal user interface.

---

##  Project Description

The Task Manager App is designed to improve productivity by allowing users to organize their daily activities in one place. It provides essential task management functionalities in a simple and intuitive Android application.





## 🛠️ Technologies Used

- Java  
- Android Studio  
- XML Layouts  
- SQLite Database  
- Material Design Components
